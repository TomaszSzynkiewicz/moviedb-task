## moviedb-task

Clone the project:
```
git clone https://github.com/TomaszSzynkiewicz/moviedb-task.git
```

cd into folder and install dependencies
```
cd moviedb-task && npm install
```

To run:
```
npm start
```

To test:
```
npm test
```

Final thoughts:
```
For the sake of simplicity pagination has been ignored for movie search.
```
